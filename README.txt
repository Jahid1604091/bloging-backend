#### Features #####
Because users feel the need to access the latest news and events, this app will be definitely helpful. 
The app will allow users to publish their own news or articles in the app itself very easily. 
Also, the users can add photos to their articles to make them look more attractive

    Login/Signup to the application
    Create articles
    View list of all the articles
    Edit/Delete his/her own articles
    Add like/comment to the articles

    Add categories, tags and filters to list blog page
    Follow other article authors
    

#### setup ####
1. create a .env file
2. npm i 
3. npm run server (dev mode)
